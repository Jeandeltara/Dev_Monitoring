# Automated Regional Media Monitoring System

## English
This project is an automated news monitoring system designed for communication units to track regional media coverage efficiently. Traditional search engine queries often yield high rates of false positives, necessitating manual review of hundreds of links. This tool automates the process by specifically scanning a curated list of regional news sites in Rivne and Dubno, filtering for relevant unit mentions.

### Key Features
* **Scheduled Monitoring:** Automatically crawls pre-defined regional news websites at set intervals.
* **Intelligent Filtering:** Searches for specific unit identifiers ("23 інженерно", "А0451") in various grammatical cases and variations.
* **AI-Powered Analysis:** Uses Grok to analyze content, providing:
    * Article title and URL.
    * Concise summaries.
    * Sentiment and tone analysis regarding the 23rd Engineer Battalion.
* **Automated Reporting:** Generates analytical reports and delivers them via email.

## Українська
Цей проєкт — система автоматизованого моніторингу регіональної преси, розроблена для комунікаційних підрозділів з метою ефективного виявлення інформаційних загроз та взаємодії зі ЗМІ. Прямий пошук через пошукові системи дає забагато хибних результатів, тому ця система автоматизує процес перегляду та фільтрації новин.

### Основні можливості
* **Автоматичний моніторинг:** Регулярний обхід визначених новинних ресурсів регіонів Рівного та Дубна за розкладом.
* **Інтелектуальна фільтрація:** Пошук специфічних згадок підрозділу ("23 інженерно", "А0451") в усіх падежах та варіаціях.
* **Аналітика за допомогою ШІ:** Використання Grok для аналізу публікацій, що включає:
    * Заголовок та посилання.
    * Короткий зміст матеріалу.
    * Оцінку настрою та тону публікації щодо 23-го інженерно-позиційного полку.
* **Звітність:** Генерація аналітичних звітів та автоматичне надсилання на електронну пошту.

### Джерела новин
Система моніторить низку регіональних ресурсів, включаючи rivnepost.rv.ua, rivne.media, ogo.ua, charivne.info, rivne.rayon.in.ua, suspilne.media, tomat.rv.ua, rivne1.tv, rp.rv.ua, itvmg.com, radiotrek.rv.ua, dubno.rayon.in.ua, teza.tv, horyn.info.
